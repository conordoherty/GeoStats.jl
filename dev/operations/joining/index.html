Joining

Overview

Multiple spatial objects can be joined into a single object:

Base.join — Method
join(data₁, data₂, joiner)

Join spatial data data₁ and data₂ with joiner method.

Methods

GeoStatsBase.VariableJoiner — Type
VariableJoiner()

Join spatial data with shared domain to produce spatial data with all variables included.
